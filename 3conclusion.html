На цій лабораторній роботі я придбала практичні навички роботи верстки сторінок засобами CSS та FLEXBOX, верстки на основі плаваючих елементів та блокової верстки.

Таблична верстка
Переваги: простота для табличних даних, легкість у розумінні та сумісність з усіма браузерами. Недоліки: погана адаптивність, складність у кастомізації, збільшення обсягу HTML-коду та ускладнення його структури, погана підтримка семантики.

Верстка за допомогою блоків (плаваючі блоки, CSS Flexbox)
Переваги: адаптивність, гнучкість, семантичність, менший код. Недоліки: складність для новачків, проблеми з підтримкою старих браузерів, нестабільність плаваючих елементів.
